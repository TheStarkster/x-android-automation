# X UI Automate

Twitter/X automation on Android via uiautomator2.

## Setup

```bash
pip install -r requirements.txt
```

Android device with USB debugging (or emulator). For `commenter.py`, set `GEMINI_API_KEY` in the environment.

## Run

```bash
python start_here.py   # basic flow
python scrapper.py     # scrape tweets
python commenter.py    # comment on tweets
```
